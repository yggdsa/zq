#蜜丝婷大草莓和小草莓对比

林清云也是低声道：“李公子，今日就此别过，妾身明日再来聆听您的教诲。”
　　“嗯，拜拜。”
　　李念凡洒脱的挥挥手。
　　他看了一眼时辰，迈步向回走。
　　刚走到城门口，就一位书生拦下。
　　书生的书生气很重，孱弱书生这个词似乎就是为他量身定制的，对着李念凡行礼道：“在下孟君良，见过公子。”
　　“孟公子你好，我叫李念凡。”李念凡疑惑的看着他，“不知道孟公子所为何事？”
　　孟君良目光炯炯的看着李念凡，“我想请教公子，是否真有长生之道？”
　　李念凡看着面前的书生，眉头微微一挑，怎么也想不到他会提这种问题。
　　这书生满身都是书生气，跟他遇到过的修仙者不同，应该只是个凡人。
　　凡人追求长生虽然也可以理解，但显然很不靠谱。
　　不瞒你说，我自己也想长生啊！
　　他试图打消这位书生的执念，开口劝道：“天道有轮回，长生之道不可为。”
　　“可是我从书上看到仙人可以长生。”
　　书生执着道：“为什么仙人可以，凡人就不可以？我找遍了所有的书籍，都没有发现长生之道，直到听到了你的故事，你一定知道长生之道，求你教我！”
　　这书生八成是读书读傻了，活在一个理想状态。
　　李念凡摇了摇头，开口问道：“仙人可以靠修仙保持自己的身体机能稳定，但凡人不行，你了解过凡人吗？”
　　书生愣住了。
　　他自己就是凡人，但却一点也不了解。
　　一直以来，他都沉浸在书的世界，看到的都是书中的知识，所有的一切都是停留在理论层面，更多的是靠自己空想！
　　他知道很多东西，但同时，又什么都不知道。
　　用李念凡前世的话说，这书生有些偏向于唯心主义。
　　只要我不想死，我就死不了。
